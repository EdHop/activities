# Activities Site

This is a page created to generate some randomised light exercise prompts.

The preview can be accessed using https://edhop.github.io/activities/

Current WIP: Creating user accounts using a supabse PostgreSQL backend to manage user details.

Planned functionality: Users can edit which activities they would like to see, add their own and amend the timings/rep ranges. This information will be stored against each user account.
